修复说明
========

原脚本使用整段文本精确匹配，pages.json 的换行符或缩进稍有不同就会提示：
“未找到 club/claimed/index 路由锚点”。

修正版改为：
1. 使用正则定位 club/claimed/index 页面对象；
2. 不依赖 CRLF/LF 和固定缩进；
3. 写入前自动验证 pages.json JSON 语法；
4. 已存在路由时自动跳过，不会重复添加。

使用：
1. 将两个脚本复制到 delta-vanguard-uniapp 项目根目录；
2. 双击 应用UniApp第三阶段第一批_修正版.bat；
3. 执行 git diff --check 和 git status --short。
